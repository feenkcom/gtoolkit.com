      <div class="col-lg-8"> 
        <p class="lead">To leverage Glamorous Toolkit, you need to <span class="manual-underline">program</span> it. The best way to learn that is to do it from within the environment. The distribution contains thousands of contextual tools that we used to develop and explain the environment itself. Learning how these tools work first accelerates your overall learning. Each of these is based on an underlying language made out of visual and interactive operators that can be combined in many ways. So, <span class="manual-underline">first learn the underlying language</span>.
        </p>
        <p class="lead">
          <a href="https://youtube.com/playlist?list=PLfrs5bwLJOoAaHvQGSLeKpHWmFuZXPUTJ" class="btn btn-lg btn-default btn-external">Watch introductory 7-minute videos on YouTube</a>

      </div>

    <div class="row padding-top-medium">
      <div class="col-lg-8"> 
        <p class="lead">The environment ships with an elaborate <span class="manual-underline">live book</span> that documents it. The book offers concrete starting steps through selected introductory videos, hands-on tutorials such as exploring CSV data, practical setup notes for working with Git, and guidance for setting up LLM connections.
        </p>
        <p class="lead">
          <a href="https://book.gtoolkit.com/get-started-ejn67l0sdt6csob78xjfb8kf4" class="btn btn-lg btn-default btn-external">Get more starting links in the book</a>
        </p>
      </div>
    </div>
    <div class="row padding-top-medium">
      <div class="col-lg-8">
        <p class="lead">Moldable Development is about adapting the environment to match the context. It's one thing to see it on someone else's problem, and a whole other experience when you apply it to your problem. We encourage you to <span class="manual-underline">pick a problem you care about</span> and work your way through it. Ideally pick a reading problem related to understanding various artifacts.</p>
